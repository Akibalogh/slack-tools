All the pools are listed on the Nitro Pools app page
If the token is not whitelisted by us, then users will have to add a contract address to the exchange to see it on the nitro pools interface
REF -> https://app.camelot.exchange/nitro

Here is a step-by-step summary of how user can add liquidity to Camelot and deploy a permissionless Nitro pool.

1) To add your liquidity
- Go to https://app.camelot.exchange/liquidity/
- Select """"LP only""""
- Select ''Your token'' and XYZ token
- Approve + add your liquidity (be careful with the rate since you will be the one to define it)

2) To create a Nitro pool
- Go to https://app.camelot.exchange/nitro/create
- Select ''Your token'' and XYZ token and click on """"Create""""
- Select XYZ as the incentive token #1
- Select the start time (set it at least to a couple of hours from the current time)
- Select the end time (you can set it +1 month from the start, it can be set to a later date any time)
- Optionally set a description if you want to
- You also have the possibility to add custom requirements (lock duration, lock end time, amount...)
- Create your Nitro

3) Deploy the Nitro pool
- Go to https://app.camelot.exchange/nitro/my, you should see your newly created Nitro
- Click on it to unroll its settings
- You can use the """"Add"""" button in the incentives section to add your XYZ incentives (you can add more anytime)
- Once you're satisfied, you can publish it (= it will be visible by everyone and open to deposits; rewards will start at the """"Start time"""")
- Everyone will have access to it on https://app.camelot.exchange/nitro/
- To share it with your community you can just open it and share the link of the page
- Be careful: you can't deploy a Nitro AFTER its start time, in that case you would need to repeat the process from the start w/ a new Nitro

Once created, you can add rewards anytime, and also extend the duration of the pool (when doing so, the remaining rewards distribution will simply be based on the new remaining duration)

4) To stake on a Nitro pool as a user
- Go to https://app.camelot.exchange/liquidity
- Check that """"Position"""" is selected
- Select ''Your token'' and XYZ, approve, and add liquidity
- Go to your XYZ-XYZ Nitro page, you should now have at least one """"compatible position"""" on the bottom
- Click on deposit
- You'll now start earning rewards

Create nitro - resources:

Document -> https://docs.camelot.exchange/protocol/nitro-pools/create-your-nitro

Guide with instructional video -> https://docs.camelot.exchange/get-started/get-started-on-camelot/nitro-pools/create-a-nitro

Hope that helps
