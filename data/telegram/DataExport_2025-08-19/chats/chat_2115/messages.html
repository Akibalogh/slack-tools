To briefly recap - we're launching dlcBTC, a safer wrapped Bitcoin. dlcBTC leverages a "smart contract" feature already in Bitcoin ‚Äî Discreet Log Contracts (DLCs) ‚Äî to enable BTC to be locked in a special type of multisig UTXO. The key feature of the DLC is that it only pays out to the original depositor, so the BTC cannot be stolen in a hack
11:38
We are not a Bitcoin L2, but we operate a set of off-chain servers called "DLC Attestors" that relay signals between BTC chain and ETH chain
11:39
Our attestors can report data on which UTXO is a DLC
11:39
When minting dlcBTC, our attestors report which UTXO is a DLC ‚Äî then anyone can check on BTC chain that it has been funded ‚Äî which mints the dlcBTC backed by Proof of Reserves
11:40
The dlcBTC then is used by HFs/traders/etc to maintain positions in AAVE/Curve/etc. So, you could show proof of reserves for those positions
