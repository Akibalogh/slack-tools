# Customer Group Audit Reports

This directory contains Excel reports generated by the customer group audit tool.

## Report Format

**Filename**: `customer_group_audit_YYYYMMDD_HHMMSS.xlsx`

Each report contains:
- All Slack and Telegram customer groups
- Team member presence analysis
- Group categorization
- iBTC rename flags
- Privacy status

## Report Columns

- **Platform**: Slack or Telegram
- **Group Name**: Name of the customer group
- **Category**: BD Customer, Marketing, Internal, or Intro
- **Requires Full Team**: Yes or No
- **Needs Rename (iBTC)**: ⚠️ YES if contains "iBTC"
- **Privacy Status**: Private or ⚠️ PUBLIC
- **History Visibility**: For Telegram - Visible, ⚠️ HIDDEN, or Unknown. N/A for Slack.
- **Total Members**: Total member count
- **Required Present**: Names of required members present
- **Required Missing**: Names of required members missing
- **Optional Present**: Names of optional members present
- **Optional Missing**: Names of optional members missing
- **Completeness**: Score like "5/5 required"

## Generate New Report

```bash
python3 scripts/customer_group_audit.py
```

Reports are automatically saved to this directory.

## Retention

- Keep latest 5 reports for historical comparison
- Delete older reports to save space
- Reports are git-ignored (not committed to repository)

## Latest Report Location

Check this directory for the most recent file:
